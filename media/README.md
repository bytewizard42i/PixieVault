# Media Folder

This folder contains media files for Pixie Vault.

## Video Files

Place your video files here to be displayed in the main window video area.

Supported formats:
- MP4 (recommended)
- AVI
- MOV
- WebM

## Usage

The video player will automatically look for video files in this folder and play them in the video area of the main window.

To add a video:
1. Copy your video file to this `media/` folder
2. The app will detect and play it automatically

## Example

```
media/
├── README.md
├── demo.mp4
└── background.mp4
```
